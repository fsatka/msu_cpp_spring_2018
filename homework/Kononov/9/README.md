
### Практическая часть

##### Ping pong

Классическая задача.

Два потока по очереди выводят в консоль сообщение. Первый выводит ping, второй выводит pong. Количество повторов - 1 000 000.

Вывод:

```
ping
pong
ping
pong
ping
pong
…
```
  
Тесты проходит, запуск:  
g++ 9.cpp -o 9 -std=c++11 -pthread && ./9
