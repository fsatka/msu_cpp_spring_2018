# Homework 02
Count amount of prime numbers in array when numbers are from [l, r] segment.

## Building and running
To build project:
```
make
```

To run project:
```
./slow   l1 r1  l2 r2  ...
./normal l1 r1  l2 r2  ...
./fast   l1 r1  l2 r2  ...
```

## Cleaning directory
```
make clean
```

## Results
| slow    | normal  | fast    |
|---------|---------|---------|
| 0.4094s | 0.0132s | 0.0033s |
